more information on dates, colors and icons
All dates need to be formatted in the same way. A few examples of dates that are valid and will be parsed correctly: 31-12-2014, 31/12/2014, 12/31/2014, 2014/12/31, 31-12-14. Times can formatted as '16:00', '16:00:00', '4:00 PM' or '4:00:00 PM'.

Durations need to be formatted like this 7d, 8.5h or 10m. Meaning 7 days, 8.5 hours or 10 minutes. The minimum duration for a task in Tom's Planner is 2.5 minutes.

The colors of periods can be set through RGB color codes (for instance: '#4286f4'). More info on RGB color codes can be found here.

All icons in the chart can imported through codes. An overview of these codes can be found here.

What is 'lock'?
When you open a schedule it's automatically locked by you. No one else can make any changes while you are working on it.

So now what?
You can 'keep the lock' and continue working. The person who sent the request will not be able to work on the schedule while you are working on it. Or you can 'unlock your schedule', which means that the user that sent you the request can start working on it. Your latest changes will be saved automatically prior to releasing the lock. After you released the lock you won't be able to make any changes to the schedule anymore.

Note: if you don't respond to this request within 60 seconds your latest changes will be saved automatically and the lock will be released.

Zooming out
When you zoom out some details can disappear.

This is a schedule with an hourly scale and four activities. Each activity takes two hours.

After zooming out to a daily scale the duration of each activity will be rounded up to half a day (the minimum step size on a daily scale). The small time block will overlap and two of them will disappear from view.
